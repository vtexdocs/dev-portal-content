---
title: "Implementing storefront features"
excerpt: "Explore prebuilt FastStore storefront features that combine UI components and best practices to implement navigation, search, regionalization, and more."
---

Storefront features are solutions that help you implement ecommerce functionality in your FastStore project. These features combine multiple [UI components](https://developers.vtex.com/docs/guides/faststore/components-index) and best practices to enhance the shopping experience for shoppers.
