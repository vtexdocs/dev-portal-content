## Overview

This feature aims to group the main store's page header components. These components perform specific functions such as navigation among the store pages, navigation to the login flow and others.

---

## Usage

We recommend the usage of the components together, keeping the `Navbar` as the wrapper of the others.

There is an available example of the usage of all the related components in our starter store [here](https://starter.vtex.app/).
