## Usage

### Import the component

```tsx
import { Icon } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/atoms/Icon/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

## Design tokens

The `Icon` component does not expose dedicated design tokens. Customize its appearance using:

- SVG props (such as `stroke`, `fill`, `width`, `height`). The visible stroke width is derived from the [`weight`](#weight) prop (`thin`, `light`, `regular`, `bold`).
- CSS targeting the `[data-fs-icon]` data attribute for layout adjustments. The default theme applies only structural styles to this selector.

## Best practices

### ✅ Do's
- Ensure icons are visually consistent in weight and size within the same context.
- Provide sufficient contrast when icons convey meaning against varied backgrounds.
- Pair icons with text when meaning may be ambiguous.

### ❌ Don'ts
- Don't rely solely on color or the icon graphic to convey critical meaning.
- Don't mix multiple icon weights within the same control or pattern.
