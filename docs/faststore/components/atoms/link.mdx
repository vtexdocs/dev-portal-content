The `Link` component behaves like a regular anchor tag (`a`), but it can be rendered as any HTML tag or component, including specific framework link components.

## Usage

### Import the component

```tsx
import { Link } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/atoms/Link/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

## Data attributes

You can target and override `Link` styles using the following data attributes:

`data-fs-link`

`data-fs-link-variant="default" | "display" | "inline"`

`data-fs-link-size="small" | "regular"`

`data-fs-link-inverse="true"`

---

## Props

The `Link` component supports all attributes supported by the `a` tag.
If you need to use it as an external link component (i.e., Next.js `Link` or Gatsby `Link`), the FastStore UI `Link` component can be used with the `as={NextLink}` prop.
