The `Input` component renders a single-line text field for capturing short user input, serving as a means of collecting information from users. Typical use cases include:

- Collecting user information (e.g., name, email, phone number).
- Search queries and filters.
- Form submissions and data entry.

## Usage

### Import the component

```tsx
import { Input } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/atoms/Input/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
