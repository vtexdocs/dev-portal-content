The `Price` component displays the price of a given product, its discount, and total values. Typical use cases include:

- Showing product prices on listings and detail pages (e.g., current selling price, original listing price).
- Displaying discounted prices and savings amounts to highlight promotions.

## Usage

### Import the component

```tsx
import { Price } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/atoms/Price/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

### INTL Formatted to parts

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <Price
        formatter={function useIntlPartsFormatter(price) {
            return useMemo(() => {
              return new Intl.NumberFormat('en-US', {
                style: 'currency',
                currency: 'USD',
              })
                .formatToParts(price)
              .map((part) => {
                const props = {
                  [`data-fs-price-${part.type}`]: true,
                }

                if (part.type === 'integer') {
                  props.style = { fontWeight: 700 }
                }

                return (
                  <span key={part.type} {...props}>
                    {part.value}
                  </span>
                )
              })
          })
        }}
        value={62.5}
        variant="selling"

    />
    </OverviewSection>

  </Tab>

### Custom price formatter

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <Price
        formatter={function customFormatter(price) {
          const unformattedPrice = `${price}`
          const formattedPrice = `${unformattedPrice.replace('.', ',')} reais`
          return formattedPrice
        }}
        value={62.5}
        variant="spot"
      />
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    import { Price } from '@faststore/ui'
    
    function customFormatter(price: number) {
      const unformattedPrice = `${price}`
      const formattedPrice = `${unformattedPrice.replace('.', ',')} reais`

      return formattedPrice
    }

    export function PriceExample() {
      return (
        <Price
          formatter={customFormatter}
          value={62.5}
          variant="spot"
        />
      )
    }
    ```

  </Tab>
</Tabs>

You can target and override `Price` styles using the following data attributes:

`data-fs-price`

`data-fs-price-variant="selling| 'listing' | 'spot' | 'savings'| 'installment"`
