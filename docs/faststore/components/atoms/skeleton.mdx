This feature improves the store's experience and perceived performance through loading placeholders.
`skeletons` are like an element's or section's wireframe. They're placeholders
that simulate the layout of these elements or sections while data loads.

Instead of showing a blank page, displaying `Skeletons` while loading content makes the user feel like the store is more responsive and faster.

Additionally, by using `skeletons`, the store will be less likely to experience [Cumulative Layout Shift (or CLS)](https://web.dev/i18n/en/cls/).

## Usage

### Import the component

```tsx
import { Skeleton } from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/atoms/Skeleton/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
