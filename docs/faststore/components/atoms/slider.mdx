The `Slider` component provides an interactive way for users to select a single value from a defined range. Common use cases include adjusting prices, setting volume, or selecting values where precision within boundaries matters.

## Usage

### Import the component

```tsx
import { Slider } from "@faststore/ui";
```

### Import Styles into your FastStore project

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/atoms/Slider/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

### Accessible Slider with `aria-value-text`

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <Slider

### Slider with custom value labels

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <Slider
        min={{ absolute: 0, selected: 10 }}
        max={{ absolute: 100, selected: 90 }}
        absoluteValuesLabel={{ min: '0', max: '100' }}
        minValueLabelComponent={(min) => <span>{min} kg</span>}
        maxValueLabelComponent={(max) => <span>{max} kg</span>}
      />
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    import React from "react";
    import { Slider } from "@faststore/ui";

    function CustomLabel({ value }: { value: number }) {
      return <span>{value} kg</span>;
    }

    export default function CustomLabelSlider() {
      return (
        <Slider
          min={{ absolute: 0, selected: 10 }}
          max={{ absolute: 100, selected: 90 }}
          absoluteValuesLabel={{ min: "0", max: "100" }}
          minValueLabelComponent={(min) => <CustomLabel value={min} />}
          maxValueLabelComponent={(max) => <CustomLabel value={max} />}
        />
      );
    }
    ```
  </Tab>
</Tabs>

## Data attributes

You can target and override Slider styles using the following data attributes:

`data-fs-slider`

`data-fs-slider-absolute-values`

`data-fs-slider-range`

`data-fs-slider-thumb="left" | "right"`

`data-fs-slider-value-label="min" | "max"`
