The `InputField` is a text field component to get user inputs. The component wraps an [Input](https://developers.vtex.com/docs/guides/faststore/atoms-input) and its corresponding [Label](https://developers.vtex.com/docs/guides/faststore/atoms-label).

## Usage

### Import the component

```tsx
import { InputField } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/molecules/InputField/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

`data-fs-input-field-error`

`data-fs-input-field-error-message`
