The `NavbarLinks` component displays a set of navigation links that self adapts on mobile or desktop screens.

This component is part of the [Navigation](https://developers.vtex.com/docs/guides/faststore/feature-navigation) feature and must be used in the [Navbar](https://developers.vtex.com/docs/guides/faststore/organisms-navbar) context.

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import {
  NavbarLinks,
  NavbarLinksList,
  NavbarLinksListItem,
} from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/molecules/NavbarLinks/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.

---

## Usage

> ⚠️ The mobile version can be achieved with
  [NavbarSlider](https://developers.vtex.com/docs/guides/faststore/organisms-navbar-slider) as wrapper.

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection containerStyle={{ display: 'block' }}>
      <Navbar>
        <NavbarLinksUsage />
      </Navbar>
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    const links = ['Office', 'Home Appliances', 'Computer and Software', 'Technology']

    <Navbar>
      <NavbarLinks>
        <div>
          <Button variant="tertiary" size="small">
            Set location
          </Button>
          <NavbarLinksList>
            {links.map((link) => (
              <NavbarLinksListItem key={link}>
                <Link variant="display" href="#">
                  {link}
                </Link>
              </NavbarLinksListItem>
            ))}
          </NavbarLinksList>
        </div>
      </NavbarLinks>
    </Navbar>
    ```

  </Tab>
</Tabs>
