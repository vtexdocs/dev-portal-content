Combine `SearchHistory`, `SearchTop`, `SearchAutoComplete` and `SearchProducts` to show all options in `SearchInput`. This component is part of the [Search](https://developers.vtex.com/docs/guides/faststore/feature-search) feature.

## Usage

### Import the component

```tsx
import { SearchDropdown } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```tsx
import "@faststore/ui/src/components/molecules/SearchDropdown/styles.scss";
```
