The `RadioField` wraps a [Radio](https://developers.vtex.com/docs/guides/faststore/atoms-radio) input and its corresponding [Label](/componentes/atoms/label). It allows users to select one option from a set.

## Usage

### Import the component

```tsx
import { RadioField } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/atoms/Radio/styles.scss'
```

```scss
@import '@faststore/ui/src/components/molecules/RadioField/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
