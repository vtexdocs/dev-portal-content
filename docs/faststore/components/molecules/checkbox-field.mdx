The `CheckboxField` wraps a [Checkbox](https://developers.vtex.com/docs/guides/faststore/atoms-checkbox) input and its corresponding [Label](https://developers.vtex.com/docs/guides/faststore/atoms-label). It allows users to select one option from a set.

## Usage

### Import the component

```tsx
import { CheckboxField } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/molecules/CheckboxField/styles.scss'
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
