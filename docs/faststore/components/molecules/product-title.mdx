The `ProductTitle` is a component commonly used on Product Details Page (PDP). It wraps the product name, a product label and the rating.

<Tabs items={["Example", "Code"]} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <ProductTitle
        title={<h1>Apple Magic Mouse</h1>}
        label={<DiscountBadge size="big" listPrice={100} spotPrice={90} />}
        refNumber="99995945"
        ratingValue={4}
      />
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    <ProductTitle
      title={<h1>Apple Magic Mouse</h1>}
      label={<DiscountBadge size="big" listPrice={100} spotPrice={90} />}
      refNumber="99995945"
      ratingValue={4}
    />
    ```
  </Tab>
</Tabs>

---

## Usage

### Import the component

```tsx
import { ProductTitle } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/molecules/ProductTitle/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
