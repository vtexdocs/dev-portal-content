The `OrderSummary` provides a summary of the items in the cart, including the total price, related shipping tax, and discounts.

        subtotalLabel="Subtotal (3 products)"

        discountValue={`-${useFormattedPrice(100)}`}
        totalValue={useFormattedPrice(1100)}
      />
    </OverviewSection>

## Usage

### Import the component

```tsx
import { OrderSummary } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/molecules/OrderSummary/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
