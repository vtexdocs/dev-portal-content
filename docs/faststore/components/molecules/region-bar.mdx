The `RegionBar` is a custom button, usually used only on mobile devices. It should trigger the [RegionModal](https://developers.vtex.com/docs/guides/faststore/organisms-region-modal).

The component is part of the [Regionalization](https://developers.vtex.com/docs/guides/faststore/feature-regionalization) feature.

## Usage

### Import the component

```tsx
import { RegionBar } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/molecules/RegionBar/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
