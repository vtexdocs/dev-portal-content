The `RatingField` component allows users to submit ratings when submitting product reviews. It wraps the [Rating](https://developers.vtex.com/docs/guides/faststore/molecules-rating) component, which displays the rating system, and the associated [Label](https://developers.vtex.com/docs/guides/faststore/atoms-label) component, which provides a text description for the rating field.

## Usage

### Import the component

```tsx
import { RatingField } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/molecules/RatingField/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
