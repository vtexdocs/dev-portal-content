Responsible for showing the latest terms searched by a user. This component is part of the [Search](https://developers.vtex.com/docs/guides/faststore/feature-search) feature.

## Usage

### Import the component

```tsx
import { SearchHistory, SearchHistoryTerm } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/molecules/SearchHistory/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
