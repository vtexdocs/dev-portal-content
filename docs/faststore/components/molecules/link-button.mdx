The `LinkButton` component is similar to a button, however acts as a link to navigate users between pages.

<Tabs items={["Example", "Code"]} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <LinkButton href="#">Link Button</LinkButton>
    </OverviewSection>
  </Tab>
  <Tab>
    ```tsx
    <LinkButton href="#">Link Button</LinkButton>
    ```
  </Tab>
</Tabs>

---

## Usage

### Import the component

```tsx
import { LinkButton } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/molecules/LinkButton/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
