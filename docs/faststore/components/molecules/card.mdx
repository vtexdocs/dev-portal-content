## Usage

### Import the component

```tsx
import { Card } from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss copy
@import "@faststore/ui/src/components/molecules/Card/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

## Data attributes

You can target and override `Card` styles using the following data attributes:

- `data-fs-card`

- `data-fs-card-header`

- `data-fs-card-title`

- `data-fs-card-body`
