FastStore UI is designed and built following the [Atomic Design](https://bradfrost.com/blog/post/atomic-web-design/) pattern.
Atomic Design breaks user interfaces hierarchically into smaller and simpler components, and the FastStore components are classified as **atoms**, **molecules**, **organisms**, and **features**.
