The `ImageGallery` is used when you want to display a series of photos in a gallery on a post or page.According to the quantity of `Image` to be displayed, the `ImageGallerySelector` will be displayed.

> ⚠️ Add the `data-fs-image` attribute in all `<img>` tags to ensure they work as expected. You can find an example in the `Code` tab below for reference.

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <ImageGalleryUsage images={imagesUsage} />
    </OverviewSection>
  </Tab>
  <Tab>
```tsx
const ImageComponent = ({ url, alternateName }) => (
  <img data-fs-image src={url} alt={alternateName} />
)

const [selectedImageIdx, setSelectedImageIdx] = useState(0)
const currentImage = imagesUsage[selectedImageIdx]

<ImageGallery
  images={imagesUsage}
  ImageComponent={ImageComponent}
  selectedImageIdx={selectedImageIdx}
  setSelectedImageIdx={setSelectedImageIdx}
>
  <ImageGalleryViewer>
    <img
      data-fs-image
      src={currentImage.url}
      alt={currentImage.alternateName}
    />
  </ImageGalleryViewer>
</ImageGallery>
```

  </Tab>
</Tabs>

## Usage

### Import the component

```tsx
import {
  ImageGallery,
  ImageGallerySelector,
  ImageGalleryViewer,
} from "@faststore/ui";
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import "@faststore/ui/src/components/organisms/ImageGallery/styles.scss";
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

## Data attributes

You can target and override `ImageGallery` styles using the following data attributes:

`data-fs-image-gallery`

`data-fs-image-gallery="with-selector | without-selector"`

`data-fs-image-gallery-selector`

`data-fs-image-gallery-selector-control`

`data-fs-image-gallery-selector-control-button`

`data-fs-image-gallery-selector-elements`

`data-fs-image-gallery-selector-thumbnail`
