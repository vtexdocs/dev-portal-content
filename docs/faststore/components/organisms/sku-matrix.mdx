The SKUMatrix simplifies product selection by allowing customers to choose and purchase different variations, like size and color, simultaneously. On a single page, they can view all options, enter quantities, and add them to their cart. The component is a compound of the following:

- `SKUMatrix`: Wraps the SKUMatrix component.
- `SKUMatrixSidebar`: Displays the table containing all the productâ€™s variation and allows the selection of multiple SKUs.
- `SKUMatrixTrigger`: Triggers the slider slider opening/closing trigger.

---

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import { SKUMatrix, SKUMatrixSidebar, SKUMatrixTrigger } from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/SKUMatrix/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.
