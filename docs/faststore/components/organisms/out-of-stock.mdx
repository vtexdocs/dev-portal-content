This `OutOfStock` component is used to announce to users that the product is out of stock. It also asks for their contact to let they know when the product arrives.


<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <OutOfStock
        title="Out of Stock"
        subtitle="Notify me when available"
        inputLabel="Email"
        buttonLabel="Notify Me"
        onSubmit={()=>{}}
        disabled={false}
      />
    </OverviewSection>
  </Tab>
  <Tab>

  </Tab>
</Tabs>

## Usage

### Import the component

```tsx
import { OutOfStock } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/OutOfStock/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).
