The `PriceRange` component is a [Slider](https://developers.vtex.com/docs/guides/faststore/atoms-slider) that allows users to select a maximum and minimum price from a range.

## Usage

### Import the component

```tsx
import { PriceRange } from '@faststore/ui'
```

### Import styles

To apply styles, include the following in your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/PriceRange/styles.scss';
```

For details, see [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles).

---

## Data attributes

You can target and override `PriceRange` styles using the following data attributes:

`data-fs-price-range`

`data-fs-price-range-inputs`
