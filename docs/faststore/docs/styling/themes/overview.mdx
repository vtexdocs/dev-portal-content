By default, every FastStore project comes with the [Brandless theme](https://developers.vtex.com/docs/guides/faststore/using-themes-overview#brandless-and-the-theming-structure). However, you have the option to use one of the [available themes](#available-themes) or [create a custom theme](#creating-a-custom-theme).

> ℹ️ You have the flexibility and freedom to style the theme by overriding it using global or local tokens, or by starting from scratch using your preferred styling tool.

## Creating a Custom Theme

1. Open your starter project and create a new `.scss` file on `src/themes` (e.g. `my-theme.scss`).
2. Add the tokens you want to overwrite inside the `.theme` class.

> ℹ️ If you want, you can work on `custom-theme.scss` file that's ready to use. We are using [CSS cascade layers](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Cascade_layers) feature to better deal with styling specificities issues, so keeping the `@layer theme` in the file is recommended.

```scss src/themes/my-theme.scss

@layer theme {
  .theme {
    // --------------------------------------------------------
    // Colors (Branding Core)
    // --------------------------------------------------------

    // --------------------------------------------------------
    // Typography (Branding Core)
    // --------------------------------------------------------

    // --------------------------------------------------------
    // Spacing (UI Essentials)
    // --------------------------------------------------------

    // --------------------------------------------------------
    // Grid & Layout (UI Essentials)
    // --------------------------------------------------------

    // --------------------------------------------------------
    // Interactive Controls (UI Essentials)
    // --------------------------------------------------------

    // --------------------------------------------------------
    // Refinements
    // --------------------------------------------------------

    // --------------------------------------------------------
    // FS UI Components
    // --------------------------------------------------------
    // Add here the customizations for FastStore UI components.
  }
}
```

3. Go to `discovery.config.js` and change the `theme` from `custom-theme` to your file's name, for example:

```js discovery.config.js
theme: 'my-theme',
```
