By the end of this track, you'll have your FastStore storefront up and running. Here's an example of what you'll build: [Starter Store](https://starter.vtex.app).

In this **Getting Started** track, you’ll create and deploy a FastStore storefront at the domain `https://{accountName}.vtex.app/`. Follow each step in order to set up your environment, start your project, run it locally, understand its structure, and integrate it with the Headless CMS.
