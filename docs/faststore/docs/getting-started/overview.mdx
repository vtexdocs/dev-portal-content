[FastStore](https://developers.vtex.com/docs/guides/faststore) is a headless commerce storefront framework built on [Next.js](https://nextjs.org/). It enables you to build high-performance, customizable ecommerce websites.

## What you'll build in this track

This track guides you through building your first FastStore storefront from scratch. By the end of this track, you’ll have a production-ready storefront deployed at `https://{accountName}.vtex.app/`, fully integrated with the Headless CMS for content management.
